# Morse Code Project: PAT 2 - Subtask 1
### Overview of Morse Code
Morse code is a method of telecommunication used to encode text characters as standardized sequences of two different signal durations, called **dots** (.) and **dashes** (-). Each character (letter or numeral) is represented by a unique sequence.
### Historical Context
Morse code was developed in the early 1830s by **Samuel Morse** and **Alfred Vail**. It was created to be used with the electrical telegraph system. It revolutionized long-distance communication because it allowed messages to be sent across wires using simple electric pulses long before voice transmission was possible. It became the global standard for maritime distress signals (like SOS) and military communication for over a century.
### How the System Works
The system is built on five elements of time:
 1. **Dot:** The basic unit of time.
 2. **Dash:** Three times as long as a dot.
 3. **Gap between signals:** The space between dots and dashes within a letter is one dot long.
 4. **Gap between letters:** The space between two letters is three dots long.
 5. **Gap between words:** The space between words is seven dots long.
In C++ programming, we often represent these using ASCII characters:
 * **Dot:** . (ASCII 46)
 * **Dash:** - (ASCII 45)
### Examples of Translations
 * **A:** .-
 * **B:** -...
 * **C:** -.-.
 * **HELLO:** .... . .-.. .-.. ---
